### Ключевые метрики (ориентиры для кровельной ниши)

| Метрика | Ориентир |
|---------|---------|
| CTR (поиск) | 5–10% |
| CTR (РСЯ) | 0.5–1.5% |
| Стоимость клика (поиск) | 30–150 руб |
| Стоимость лида | 1 000–2 500 руб |
| Конверсия сайта | 3–8% |

### Советы

1. **Разделяй кампании по теплоте** — горячие запросы отдельно от информационных
2. **Гео: Волгоград + область** — Волжский, Камышин, Михайловка, Урюпинск
3. **Время показов** — с 7:00 до 21:00, пн-сб (основное время звонков)
4. **Мобильная корректировка** — +20% для мобильных (звонки)
5. **Сезонность** — пик: март-октябрь, увеличивай бюджет с апреля
6. **Ретаргетинг** — догоняй тех, кто был на сайте, но не оставил заявку
7. **Быстроссылки** — цены, отзывы, портфолио, бесплатный замер
8. **Тексты объявлений** — цифры, конкретика, гарантия, бесплатный выезд
