## СПРАВКА: ОПЕРАТОРЫ ЯНДЕКС ДИРЕКТ

| Оператор | Что делает | Пример |
|----------|-----------|--------|
| `+` | Фиксирует предлог/союз | `кровля +в волгограде` — не покажет «кровля волгоград» без «в» |
| `!` | Фиксирует форму слова | `!купить` — только «купить», не «покупка» |
| `""` | Точное количество слов | `"монтаж кровли"` — только 2 слова, без дополнений |
| `[]` | Фиксирует порядок слов | `[из волгограда в москву]` — именно этот порядок |
| `()` и `\|` | Группировка вариантов | `кровля (монтаж\|ремонт) волгоград` |
| `-` | Минус-слово к фразе | `кровля -своими -руками` |

### Рекомендация по операторам для этой ниши:

- **Используй `+` для предлогов** — иначе Директ может их игнорировать
- **Не используй кавычки `""` на старте** — сначала собери статистику по широким, потом сужай
- **Минус-фразы на уровне кампании** — экономит время, применяется ко всем группам сразу
