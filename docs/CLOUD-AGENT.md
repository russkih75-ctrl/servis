# Как запустить облачного агента (Cloud Agent) здесь

Облачной агент Cursor работает в облаке, а не на вашем компьютере: можно закрыть ноутбук — задача продолжит выполняться.

---

## Запуск из Cursor (здесь, в этом проекте)

1. Откройте **чат с агентом** (Cmd/Ctrl + L или кнопка Agent).
2. В поле ввода агента найдите **выпадающий список** (там, где выбирается режим/модель).
3. Выберите **«Cloud»** (Облако) вместо локального запуска.
4. Напишите задачу и нажмите Enter — агент запустится в облаке.

После этого агент работает на удалённой машине Cursor. Статус и логи можно смотреть в том же чате; при необходимости агент сделает коммиты в репозиторий.

---

## Окружение уже настроено в репозитории

В проекте есть **`.cursor/environment.json`**: перед каждым запуском облачного агента выполняется `npm install`. Дополнительная настройка в репо не нужна.

**Один раз сделайте привязку репо к облаку:**

1. Откройте **https://cursor.com/onboard**.
2. Подключите **GitHub** и выберите репозиторий **russkih75-ctrl/servis**.
3. При необходимости укажите секреты (API-ключи и т.п.) — **cursor.com** → Dashboard → **Cloud Agents** → **Secrets**.
4. Дождитесь завершения первой настройки; при желании сохраните снимок VM для быстрых следующих запусков.

После этого можно запускать облачного агента из чата (режим **Cloud**) — окружение подхватится из репо автоматически.
